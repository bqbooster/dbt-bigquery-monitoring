## dbt-bigquery-monitoring v0.24.0 - December 21, 2025

### Features


  - Add GitHub Actions workflow and Python script for cleaning up BigQuery test datasets

  - Compatibility with dbt Fusion is monitored through dedicated CI workflow and documentation updated for the support

### Fixes


  - Address issues for dbt Fusion compatibility

  - Comprehensive update of Information Schema models (SQL and YAML) to improve consistency and fix missing columns

### Under the Hood


  - Centralize dataset fetching logic into a get_dataset_list() macro to avoid redundant run_query calls during JIT rendering

  - Optimize dbt compile time for dbt Fusion by disabling static analysis on introspective models
