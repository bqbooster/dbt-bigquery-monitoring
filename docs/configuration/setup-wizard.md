# Setup guide

Follow these steps to configure the package before scheduling your first run.

1. Start with the [configuration decision tree](/configuration/decision-tree).
2. Validate required values in the
   [configuration matrix](/configuration/configuration-matrix).
3. Apply variables in
   [package settings](/configuration/package-settings).
4. Continue to [running the package](/running-the-package) to schedule your first run.
